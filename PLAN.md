## Steps

### Step 1 - Repository Setup

1. Create public repository on GitHub
2. Initialize with README
3. Create dev branch

### Step 2 - File Preparation

1. Finalize QMD file
2. Render PDF output
3. Ensure all required files are included
4. Use relative file paths

### Step 3 - Version Control Workflow

1. Work in dev branch
2. Create issues for tasks
3. Make multiple commits with clear messages
4. Link commits to issues

### Step 4 - Documentation

1. Complete README file
2. Add this PLAN document
3. Organize repository structure

### Step 5 - Pull Request

1. Create pull request from dev → main
2. Add title and description
3. Reference issues
4. Merge pull request

### Step 6 - Final Check

1. Ensure main branch is up to date
2. Confirm issues are closed
3. Verify reproducibility of QMD
4. Review commit history
