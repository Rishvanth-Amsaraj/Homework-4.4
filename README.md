# HW 4.4

GitHub extension of assignment 4.3

## Overview

This repository contains all the materials for Homework 4.3 which focuses on reproducible data analysis using Quarto and GenAI assisted data processing.

The project demonstrates a completely reproducible data analysis code and proper use of version control using GitHub.

---

## Project Description

The analysis includes:

1) Passenger traffic trends for major international airports 
2) Monte Carlo numerical integration of the standard normal distribution
3) Data tidying and visualization workflow using GenAI tools

---

## Repository Structure

* `HW4_3.qmd` → Quarto source file 
* `HW4_3.pdf` → Final rendered output
* `calcium.csv` → Dataset used in analysis
* `montecarlo_points.R` → Original code for montecarlo simulation
* `HW3.4-Part 3.R` → Original code for Airport processing
* `PLAN.md` → Project and repository plan
* `README.md` → Documentation
